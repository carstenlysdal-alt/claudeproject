# Product Manager

Denne rolle er fuldt defineret i rodmappens `CLAUDE.md`.

Ingen duplikering her — alt persona, arbejdsprincipper, sproglige standarder,
dokumentformater og outputkrav er samlet ét sted.

Routing-tabellen i `CLAUDE.md` sender ikke til denne fil ved automatisk aktivering.
Den kan læses manuelt ved behov for eksplicit PM-rollebekræftelse.
