# GTM & Kommerciel Bundle

Samler alle skills til go-to-market eksekvering, salg og kommerciel vækst.

## Skills i dette bundle

### Launch og GTM
| Skill | Aktivér når |
|---|---|
| `launch` | Planlægge produktlaunch i 5 faser (intern → alpha → beta → early access → GA) |
| `product-manager-skills:press-release` | Skrive pressemeddelelse eller launch-announcement |
| `growth-loops` | Identificere og designe vækst-loops (viral, referral, usage) |

### Salg og enablement
| Skill | Aktivér når |
|---|---|
| `sales-enablement` | Pitch decks, one-pagers, objection handling til Commercial Lead |
| `revops` | Revenue engine: lead lifecycle, scoring, routing, pipeline hygiejne |
| `cold-email` | B2B cold outreach og prospekt-mail |

### Indhold og SEO
| Skill | Aktivér når |
|---|---|
| `content-strategy` | Content pillars, topic clusters, redaktionel strategi |
| `seo-audit` | Teknisk SEO, ranking-problemer, on-page-optimering |
| `seo` | Generel SEO-rådgivning og søgeordsanalyse |
| `programmatic-seo` | Skalerbar SEO via skabeloner og datadrevet indhold |
| `marketing-ideas` | Generere markedsføringidéer og taktikker |

### Email og automatisering
| Skill | Aktivér når |
|---|---|
| `emails` | Email-sekvenser, drip-kampagner, onboarding-flow |
| `cold-email` | Cold outreach og B2B-prospekt-mail |

### Konvertering og analyse
| Skill | Aktivér når |
|---|---|
| `cro` | Konverteringsoptimering, landing page-audit |
| `analytics` | Analytics-opsætning, tracking-plan, GA4, GTM |
| `ads` | Paid advertising: Meta, Google, LinkedIn |
| `social` | SoMe-opslag og organisk social distribution |

## Brugsanvisning

1. Start med `launch` for det overordnede GTM-framework.
2. Brug `sales-enablement` til at forberede Commercial Lead.
3. Kobl `content-strategy` + `seo-audit` for redaktionel og organisk vækst.
4. Mål med `analytics` og optimér med `cro`.
5. Skalér med `growth-loops` og `programmatic-seo`.
